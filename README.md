# 🎮 TrueReplayer

**TrueReplayer** é um aplicativo para Windows que permite **gravar e reproduzir ações do usuário**, como cliques do mouse, rolagem e teclas pressionadas. Com uma interface moderna e recursos de automação avançados, ele é ideal para tarefas repetitivas.

## ✨ Funcionalidades

- ✅ **Gravação de Ações**
  - Registra cliques (esquerdo, direito, do meio), teclas e scroll.
  - Hotkey configurável (padrão: `F2`).

- 🔁 **Reprodução de Ações**
  - Executa ações com delays personalizados.
  - Suporte a repetição em loop com intervalo (padrão: `F3` para iniciar/parar).

- 📝 **Gestão de Ações**
  - Tabela interativa para visualizar, editar e remover ações.
  - Edição de delay direto na grid.
  - Copiar ações para a área de transferência com formatação de colunas.

- 💾 **Perfis**
  - Salve e carregue perfis `.json` contendo ações e configurações.
  - Armazena preferências como hotkeys, loop, delay e aparência.

- ⚙️ **Configurações**
  - Delay personalizado por ação.
  - Loop com contagem e intervalo.
  - Opções visuais: "Sempre no Topo", minimizar para bandeja.

---

## 🧪 Como Usar

1. **Gravar**:
   - Pressione `F2` ou clique em "Recording".
   - Realize as ações desejadas.
   - Pressione `F2` novamente para parar.

2. **Reproduzir**:
   - Pressione `F3` ou clique em "Replay".
   - As ações serão executadas conforme gravadas.

3. **Editar**:
   - Edite delays ou comentários diretamente na tabela.

4. **Salvar / Carregar Perfil**:
   - Clique em "Save" para exportar um perfil `.json`.
   - Clique em "Load" para importar um perfil existente.
